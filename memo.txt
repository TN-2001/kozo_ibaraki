  if (!window.Create(L"MyApp", origin, size)) 

-----web(Firebase)リリース-----
・Firebaseでプロジェクト作成
・npm install -g firebase-tools
・firebase login
・firebase init
　（Hosting を選択
　　Use an existing project を選択し、作成したFirebaseプロジェクトを選択
　　public directory を build/web に設定
　　single-page app として設定）
・flutter build web

・複数ページのとき
 firebase.jsonに以下を追加
    "rewrites": [
      {
        "source": "**",
        "destination": "/index.html"
      }
    ]

・firebase deploy

-----web(Firebase)更新-----
・flutter build web
・firebase deploy
